# BudgetCalc

## General info
This project is simple Budget Calculator.
	
## Technologies
Project is created with:
* Java JDK 1.8.0_211
* Gradle 5.0
* JavaFX 
* Scene Builder 2.0
* Eclipse 2019-03
	
## Setup
To run this project, install it locally using gradle:

```
$ ./gradlew build
$ cd build/libs/
$ java -jar BudgetCalc.jar
```
